# Диагностика блока управления

Страница блока управления  — это основная рабочая страница выбранного электронного блока управления. Используйте её, чтобы в одном месте получить всю информацию о блоке и выполнить доступные диагностические действия.

Чтобы открыть страницу блока управления, перейдите в раздел **Блоки управления** и выберите ЭБУ из списка, затем нажмите кнопку **Описание и функции** боковой панели.

На странице ЭБУ вы можете:
- просмотреть идентификационные данные блока;
- проверить наличие диагностических кодов неисправностей (DTC);
- перейти к детальной информации по ошибкам;
- запустить диагностические операции, тесты и рутины;
- получить доступ к технической документации и электросхемам, связанным с блоком.

Также со страницы блока управления доступны прямые ссылки на:

- техническую документацию ЭБУ;
- электросхемы;
- документацию по LIN-устройствам (если применимо).

Используйте эти материалы для углубленного анализа неисправностей и выполнения ремонтных операций.

## Диагностические коды неисправностей (DTC) блока

Чтобы считать ошибки конкретного блока управления, нажмите кнопку **Считать DTC блока** в верхней части страницы.

После выполнения операции:
- отображается список найденных DTC;
- для каждого кода показываются его статус и краткое описание;
- элементы списка подсвечиваются цветом в зависимости от статуса:
    * Активная — неисправность присутствует в данный момент;
    * Сохранённая — неисправность была зафиксирована ранее.

Если при последнем считывании ошибки не обнаружены, выводится сообщение "При чтении блока DTC не обнаружено". Если считывание DTC ещё не выполнялось, отображается соответствующее уведомление.

:::tip
Чтобы открыть подробную информацию о конкретной ошибке, нажмите на элемент DTC в списке.
:::

## Диагностические операции

На странице блока управления доступны диагностические операции для этого ЭБУ: 
- тесты исполнительных механизмов,
- мониторинг действительных значений,
- сервисные операции,
- тесты, 
- рутины.

Для выполнения диагностической операции выберите тип операции из списка в боковой панели, выберите параметры операции и следуйте инструкциям.

### Исполнительные механизмы 

Раздел **Исполнительные механизмы** предназначен для активации и проверки работы отдельных функций и механизмов, управляемых блоком.

Чтобы активировать механизм:
- Откройте раздел **Исполнительные механизмы** боковой панели на странице ЭБУ.
- Выберите механизм из списка.
- Ознакомьтесь с условиями выполнения и предупреждениями.
- Выполните доступное действие (включение, выключение, переключение режимов).
