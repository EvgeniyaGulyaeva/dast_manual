## Таблица ЭБУ

В табличном режиме для каждого ЭБУ отображаются следующие колонки:
* Код ЭБУ — краткое обозначение блока (например, BCM, BMS, EPS);
* Название — полное наименование блока (например, Body Control Module);
* Статус: 
    + В сети — ЭБУ доступен и отвечает; 
    + Не в сети — ЭБУ не отвечает; 
    + Не установлен — ?;
* Шина — тип шины, к которой относится ЭБУ (Body, Chassis, Powertrain и др.);
* Версия ПО — установленная версия программного обеспечения;
* DTC — информация о наличии диагностических кодов:
    + Считывание не проводилось;
    + Не обнаружено;
    + Коды ошибок (после считывания).

The 'ECU list' page should contain a side bar that should allow:
- to read DTC separately for each ECU
- to clean DTC separately for each ECU

## Возможные проблемы и ограничения

ECU list screen can be displayed after DAST processed information from Diagcore topology from the vehicle;

ECU list screen content depends on the data retrieved from the vehicle after completion of operations on the vehicle side (e.g. ECU/HPC status request, presence of DTC, software versions, etc.); More possible details here: UI specification \ Description by BA.

Amount of information to be displayed on ECU list screen depends on the data retrieved from the vehicle ECU/HPCs and from integrations with other systems;
