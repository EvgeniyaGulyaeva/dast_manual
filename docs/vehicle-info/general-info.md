# Основная информация об автомобиле

## для чего нужно
на главной странице отображаются основные данные о подключенном автомобиле (можно выбрать автомобиль в хедере, если подключено несколько)

## что нужно делать / знать

:::info
часть параметров недоступны при офлайн работе
:::

| Параметр | Офлайн-доступ | Примечание |
| -------- | ------------- | ---------- |
| **Статус подключения** | ✓ | |
| **VIN** | ✓ | Уникальный идентификатор автомобиля |
| **Модель / Год выпуска** | ? |  |
| **Пробег** | ✓ | Используется для расчёта интервалов ТО |
| **Дата последнего ТО** | ✓ | (откуда инфа?) |
| **Гарантия** | ✗ | Дата окончания срока гарантийного обслуживания |
| **Заряд ВВ-батареи** | ✓ | Обновляется в реальном времени |
| **Статус кабеля ВВ-зарядки** | ✓ | (возможные значения?)|
| **Емкость ВВ-батареи** | ✓ | (расшифровка и рекомендации?) |
| **Напряжение 12В сети** | ✓ |  |
| **Наличие обновлений ЭБУ** | ✗ | Наличие новых версий ПО для электронных блоков управления |

## что может пойти не так
не получены данные (все или часть) от тачки или из облака
