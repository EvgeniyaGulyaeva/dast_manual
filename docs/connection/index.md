<Admonition type="info" title="Предоставление доступа к автомобилю">
Автомобиль должен быть зарегистрирован для сервисного обслуживания на конкретной сервисной станции. Регистрация выполняется владельцем автомобиля или иным уполномоченным пользователем _на портале Atom Aftersales?_. После регистрации автомобиль становится доступным в списке автомобилей для обслуживания.

Если автомобиль отсутствует в списке, обратитесь к менеджеру сервисной станции (мастеру-консультанту, мастеру цеха).
</Admonition>

## Выбор автомобиля для подключения

Для начала работы выберите автомобиль из списка. Идентификация автомобиля осуществляется по VIN.

Сверка VIN выполняется по маркировке на автомобиле либо по данным, указанным в документации на ремонт.

    src={useBaseUrl('/img/Connect-VIN-list.png')}
    alt="Выбор VIN"
  />
  <figcaption>Выбор VIN для подключения</figcaption>
</figure>

:::info
При необходимости возможно одновременное подключение нескольких автомобилей. Чтобы выбрать автомобиль из нескольких подключенных, используйте дропдаун с VIN в хедере.
:::

## Подготовка автомобиля к подключению

Перед подключением автомобиля к системе DAST убедитесь, что
- автомобиль не находится в спящем режиме (<mark>как это проверить?</mark>),
- автомобиль переведён в сервисный режим <mark>(? Подробности пока не известны)</mark>.

| В системе отсутствуют автомобили, доступные для диагностики | <mark>никто не зарегистрировал/неправильная роль?</mark> | обратитесь к менеджеру сервисной станции? |
