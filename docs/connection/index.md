## список доступных для подключения автомобилей

:::info
**пререквизиты**

автомобиль должен быть зарегистрирован для сервисного обслуживания на конкретной сервисной станции. это делает владелец или другой пользователь автомобиля, тогда автомобиль появляется в списке доступных для обслуживания. 

если автомобиля в списке нет, куда бежать? - к менеджеру (мастеру-консультанту, мастеру цеха) СП;
:::

- диагност заходит в DAST и видит список доступных для диагностики автомобилей (от 0 до n, если 0 - значит никто свой автомобиль в AS портале не зарегистрировал и не дал к нему доступ, можно сидеть ковырять в носу? - - решение как для п.1 (автомобиля нет в списке). Также, обсуждался кейс с ручным вводом VIN и ЗН и получением доступа к DAST в «Ограниченном режиме»);
- диагност выбирает автомобиль по VIN? как это происходит? как смэтчить VIN и стоящую рядом тачку? - - по VIN на автомобиле (или в документах на ремонт);
- диагност выбирает способ подключения: по кабелю/по вайфаю? - Способ выбирается в зависимости от доступности (не факт, что на VMP будет Wi-Fi), удобства, наличия ограничений по способу подключения (некоторые операции с а/м могут потребовать определённого способа подключения, это будет указано в инструкциях к операции);

если выдаст несколько VIN - выбрать нужный (где его посмотреть? почему Aftersales портал может отдать больше одного VIN при подключении по кабелю?) - ASP выдаст список VIN, для которых у данного СП/диагноста есть разрешение на подключение. Диагност выбирает нужный VIN, сверив его с VIN на а/м (или в документах на ремонт);

## подготовка автомобиля к подключению

что нужно сделать/проверить сервисному инженеру, чтобы подготовить автомобиль? достаточно ли просто воткнуть кабель или нужно предпринять что-то еще?

Для подключения по кабелю: 1) подключить кабель; 2) автомобиль не в спящем режиме; 3) Активировать Сервисный режим в а/м (? Подробности пока не известны).
Для подключения по Wi-Fi: 1) Активировать Wi-Fi в автомобиле (? Подробности пока не известны); 2) автомобиль не в спящем режиме; 3) Активировать Сервисный режим в а/м (? Подробности пока не известны).
