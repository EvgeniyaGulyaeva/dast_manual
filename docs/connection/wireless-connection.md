# Беспроводное подключение

Беспроводное подключение позволяет работать с автомобилем без кабеля. Этот способ удобен, если кабель мешает, его длины недостаточно или требуется диагностировать автомобиль, не подходя к нему вплотную.

:::info
При необходимости возможно одновременное подключение нескольких автомобилей. Чтобы выбрать автомобиль из нескольких подключенных, используйте дропдаун с VIN в хедере DAST.
:::

<mark>как подготовить автомобиль?</mark>

:::note
Перед подключением убедитесь, что автомобиль не в спящем режиме.
:::

Чтобы подключиться к автомобилю по беспроводному соединению:

1. Выберите тип подключения **По беспроводному соединению**. 
2. Дождитесь завершения поиска доступных для подключения автомобилей.
