# Беспроводное подключение

<mark>Без кабеля может быть удобнее работать с а/м (если кабель мешает или его длины не хватает. Если автомобили подготовлены к подключению, то можно подключаться к ним и работать не вставая с места).</mark>

:::tip 
можно подключить несколько автомобилей и переключаться между ними?
:::

<mark>как подготовить автомобиль?</mark>

Выберите тип подключения **По беспроводному соединению**. DAST начнет поиск доступных для подключения автомобилей.
