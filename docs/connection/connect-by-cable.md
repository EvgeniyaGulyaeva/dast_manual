# Подключение по кабелю

## что нужно делать и/или знать

:::info
Для подключения автомобиля по кабелю необходим кабель Ethernet-OBD-II (см. раздел [Требования и меры предосторожности](/docs/getting-started/requirements.md))
:::

1. Откройте список доступных для обслуживания автомобилей. 
2. Выбрать VIN автомобиля из списка. Сверка VIN выполняется по маркировке на автомобиле либо по данным, указанным в документации на ремонт.
3. Переведите автомобиль в сервисный режим, следуя инструкциям DAST или Инструкции по эксплуатации электромобиля Атом. 
4. Вставьте кабель в разъем ...
5. Вставьте кабель в другой разъем ... 
6. успех?
