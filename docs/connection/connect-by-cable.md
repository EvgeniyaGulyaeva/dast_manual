# Подключение по кабелю

## что нужно делать и/или знать

:::info
кабель Ethernet-OBDII (см. пререквизиты)
:::

открыть список доступных для обслуживания автомобилей 

выбрать нужный по VIN на автомобиле (или в документах на ремонт);

на руле жмайте кнопки, воткните провод туда, воткните провод сюда 

<img
  src={useBaseUrl('/img/Connect-Page-Tooltip.png')}
  className="img-card"
  alt="Процесс подключения"
/>

на экране увидите прогресс 

<img
  src={useBaseUrl('/img/Connection-Process.jpg')}
  className="img-card"
  alt="Процесс подключения"
/>

начать работу 

<img
  src={useBaseUrl('/img/Connection-Success.png')}
  className="img-card"
  alt="Подключение успешно"
/>

# что может пойти не так 

Что будет, если активация режима Сервис недоступна? Например, не работает SWP.

сломался кабель? или неплотно вставлен в разъем?

ошибка чтения данных -> попробуйте снова (что делать, если ошибка сохраняется?)

:::note
низкий уровень заряда -> подключите зарядку 

есть критически низкий уровень, когда продолжать работу нельзя? 

что произойдет, если разрядится в процессе работы?
:::
