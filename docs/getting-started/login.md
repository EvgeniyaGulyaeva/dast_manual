## что может пойти не так? 

На текущий момент, только авторизация через AТОМ ID.
Если код не приходит, то в ATOM ID должна быть стандартная функция помощи.

- Проблема с интернетом
- Роль пользователя без допуска к DAST
- Сбой сервисов авторизации (ATOM ID, SPAS, PKI (?))
- Экземпляр DAST не прошёл процедуру регистрации
- Несоответствующий браузер на экземпляре DAST или некорректные настройки браузера (?)
- Настройки антивируса, брэндмауэра, VPN (?)
- Занятные порты, используемые DAST-agent
