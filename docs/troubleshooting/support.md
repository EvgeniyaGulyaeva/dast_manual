# служба поддержки атом

## когда обращаться в поддержку 
- Не смог самостоятельно устранить неисправность/рекламацию клиента (используя все имеющиеся в его распоряжении инструменты, источники информации);
- Согласование объёма/стоимости ремонта, если есть такие требования в Руководстве по Гарантии или других директивных документах Атом;
- Обязательное уведомление перед ремонтом (например, определённые неисправности, по которым Производитель проводит анализ и ему необходима дополнительная информация по рекламациям), если есть такие требования со стороны Техподдержки или отдела Качества Атом.

## контакты поддержки 

## что подготовить для обращения в поддержку
Перечень может быть большим. Основное:

- Причина обращения. С классификацией, описанием симптомов, перечнем и результатами выполненных проверок;
- Фото, видео и другая документация, относящаяся к обращению;
- Диагностический протокол со всеми выполненными проверками и их результатами;
- Калькуляции на ремонт (если это согласование объёма/стоимости);
- Другая информация по требованию Техподдержки.
