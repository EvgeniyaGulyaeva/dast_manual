# Информация об автомобиле

После успешного подключения автомобиля становится доступна информация, необходимая для диагностики и сервисного обслуживания. На главной странице DAST представлены идентификационные данные и основные параметры автомобиля, сведения о комплектации, сервисных и отзывных кампаниях, а также история обслуживания.

## Основная информация об автомобиле

В верхней части главной страницы DAST отображаются основные данные подключенного автомобиля. Если к системе подключено несколько автомобилей, нужный автомобиль можно выбрать в хедере.

<figure className="img-card">
  <img
    src={useBaseUrl('/img/general-info.png')}
    alt="Основная информация об автомобиле"
  />
  <figcaption>Основная информация об автомобиле</figcaption>
</figure>

Данные из этого раздела используются для первичной оценки состояния автомобиля и подготовки к дальнейшим диагностическим операциям.

:::warning
Часть данных недоступна при работе в офлайн режиме.
:::

**Отображаемые параметры**

| Параметр | Офлайн-доступ | Примечание |
| -------- | ------------- | ---------- |
| **VIN** | ✓ | Уникальный идентификатор автомобиля |
| **Модель / Год выпуска** | ? |  |
| **Пробег** | ✓ | Используется для расчёта интервалов ТО |
| **Дата последнего ТО** | ✓ | (откуда инфа?) |
| **Гарантия** | ✗ | Дата окончания срока гарантийного обслуживания |
| **Заряд ВВ-батареи** | ✓ | Обновляется в реальном времени |
| **Статус кабеля ВВ-зарядки** | ✓ | (возможные значения?)|
| **Емкость ВВ-батареи** | ✓ | (расшифровка и рекомендации?) |
| **Напряжение 12В сети** | ✓ | (для чего нужно?) | 
| **Наличие обновлений ЭБУ** | ✗ | Наличие новых версий ПО для электронных блоков управления (кликабельное, куда ведет?) |


## Сервисные и отзывные кампании

В разделе **Сервисные и отзывные кампании** отображается информация о кампаниях, относящихся к данному автомобилю.

Активные кампании подлежат выполнению, завершенные кампании отображаются для справки и позволяют избежать повторного выполнения работ. Информация о ранее выполненных кампаниях также учитывается при анализе рекламаций клиента.

При переходе в карточку кампании отображается ее подробное описание и связанная документация.

</figure>
